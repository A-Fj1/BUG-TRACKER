# BugTracker Pro
A full-stack Bug Tracker web app to track projects, issues, teams & comments.

## Requirements
- Python 3.x

## Setup

### 1. Install dependencies
```bash
pip3 install -r requirements.txt
```

### 2. Run the server
```bash
python3 app.py
```

### 3. Open in browser
http://127.0.0.1:5000

## Features
- Dashboard — live stats
- Users — Add / Edit / Delete
- Teams — Add / Edit / Delete
- Projects — Add / Edit / Delete
- Issues — Add / Edit / Delete
- Comments — Add / Delete

## Tech Stack
- Python · Flask · SQLite
